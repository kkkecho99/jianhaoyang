Task Introduction: Take-Out Ordering Interface

  • home

Project Overview

This project aims to enhance a basic restaurant database website into a user-friendly, responsive take-out ordering interface. The interface will allow users to browse dishes, filter options, and create and save orders.

Task Requirements

  • Reformat meals for better visual presentation on mobile and desktop screens.
  • Add filters for meals (e.g., by restaurant, vegetarian options, price range, or specific terms).
  • Allow users to create orders and save them to the database.
  • Implement a secure page to display all received orders ("orders_received.php").
  • Ensure the site is secure by enforcing password policies and preventing SQL injection attacks.
